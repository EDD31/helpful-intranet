== Description ==

GOV.UK-style intranet WordPress theme


= Content types =

* Tasks
* Guides
* Projects
* Vacancies
* News
* Blog


== Installation ==

This theme requires the Pods WordPress plugin to manage custom post types.
http://wordpress.org/extend/plugins/pods/
Tested up to: 2.3.4
